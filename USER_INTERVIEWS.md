# User Interview Logs

### Interview 1: Startup Founder (Abhishek)
- **Key Pain:** Paying for redundant seats (OpenAI vs. Cursor).
- **Insight:** Founders value "Total Burn" over individual tool cost. 
- **Action for EQ AI:** Add a "Redundancy Check" to the audit engine.

### Interview 2: Independent Developer (Mehul Chandak)
- **Key Pain:** Tier-jumping. Being forced into Team plans for message limits.
- **Insight:** Users want to know if switching to API usage is cheaper than a subscription.
- **Action for EQ AI:** Compare "Subscription cost" vs "Heavy API usage cost."

### Interview 3: Engineering Manager (Kavya)
- **Key Pain:** Invoice fragmentation and expense report overhead.
- **Insight:** Managers will pay a slight premium for centralized billing.
- **Action for EQ AI:** Recommend "Team Plans" for teams > 5 to simplify operations.
